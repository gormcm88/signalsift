# READY_TO_USE.md

A practical shortlist of curated open/free skill packs that are worth using now.

## 1. Superpowers
Location: `skills/curated/superpowers/`

Best sub-skills:
- `systematic-debugging` — use before fixing bugs
- `test-driven-development` — write failing tests first
- `verification-before-completion` — verify results before claiming done
- `writing-plans` — produce execution plans
- `executing-plans` — work through plans cleanly

Use when:
- debugging
- fixing broken code
- implementing in a more disciplined way
